## Responsive Behavior

### Mobile (< 600px)
- Header height: 200px
- Full-width details
- Single column layout
- Touch-friendly buttons (48px)

### Tablet (600px - 900px)
- Header height: 220px
- Larger icons (72px)
- More padding
- Wider detail cards

### Desktop (> 900px)
- Header height: 240px
- Max-width container (600px)
- Centered on screen
- Large text
